# 🌌 AI Multiverse Chatbot

An interactive AI chatbot built with Streamlit and Google Gemini AI that allows users to chat with multiple AI personalities such as Teacher, Programmer, Motivator, Comedian, and AI Engineer.

## 🚀 Features

- 🤖 Multiple AI Personalities
  - Teacher
  - Programmer
  - Motivator
  - Comedian
  - AI Engineer

- 💬 Interactive Chat Interface
- 🧠 Powered by Google Gemini AI
- 📝 Chat History Support
- 🗑️ Clear Chat Option
- 🎨 User-Friendly Streamlit UI
- 🔒 Secure API Key Management using `.env`

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API
- python-dotenv

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/saro0923/AI-Multiverse-Chatbot.git
cd AI-Multiverse-Chatbot
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure Gemini API Key

Create a `.env` file in the project root directory:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Get your API key from:

https://aistudio.google.com/

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will be available at:

```text
http://localhost:8501
```
